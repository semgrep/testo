Module Testo_util.Create_temp_file

Reimplementation of temporary file creation

This solves the following problems:

  • open files on Windows such that they can be deleted even if some file handles still exist, which occurs after output redirection via dup/dup2.
  • force a reinitialization of the random number generator sequence after a Unix.fork().
